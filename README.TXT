# Reminder App (Node.js + Express)

How to Run:
1. npm install
2. node src/app.js
3. Open frontend at public/index.html
4. API runs at http://localhost:4000

Features:
✔ Add reminders
✔ Edit reminders
✔ Delete reminders
✔ View all reminders
✔ Health check endpoint for CI/CD + Docker
